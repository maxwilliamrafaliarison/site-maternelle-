## 📋 Présentation

Ce site centralise toutes les ressources pédagogiques pour la classe de **Très Petite Section (TPS) et Petite Section (PS)** du Collège Français René Cassin de Fianarantsoa, Madagascar.

- **Enseignante :** Mme Fy ANDRIANARIVO
- **Classe :** TPS (8 élèves) + PS (12 élèves) = 20 élèves
- **ATSEM :** 1 personne
- **Année scolaire :** 2025 – 2026
- **Programmes :** Cycle 1 – nouveaux textes 2025 (EVAR + EDD intégrés)

## 🎯 Domaines d'apprentissage couverts

Les 7 domaines du cycle 1 (programmes 2025) :

1. **💬 Langage oral et écrit** – vocabulaire, syntaxe, phonologie, albums, pré-lecture, pré-écriture
2. **🔢 Premiers outils mathématiques** – nombres (1-6), formes, grandeurs, problèmes, motifs
3. **🏃 EPS – Activité physique** – motricité globale, parcours, jeux collectifs, danse, spectacle
4. **🎨 Activités artistiques** – dessin, graphisme, compositions plastiques, musique, chants
5. **🌍 Explorer le monde** – temps, espace, vivant, matière, outils numériques
6. **❤️ EVAR** *(nouveau 2025)* – vie affective et relationnelle, émotions, relations, règles de vie
7. **🌿 EDD** *(nouveau 2025)* – éducation au développement durable, jardinage, tri, écocitoyenneté

## 🚀 Déploiement sur GitHub Pages

### Étapes pour publier le site :

```bash
# 1. Créer un nouveau dépôt sur github.com
# Nom suggéré : maternelle-rene-cassin-2025

# 2. Initialiser git dans le dossier site-maternelle
cd chemin/vers/site-maternelle
git init
git add .
git commit -m "Initial commit - Site maternelle TPS/PS 2025-2026"

# 3. Connecter au dépôt distant
git remote add origin https://github.com/VOTRE_USERNAME/maternelle-rene-cassin-2025.git
git push -u origin main

# 4. Activer GitHub Pages
# → Aller dans Settings > Pages > Source : main branch / root
# → Le site sera accessible à : https://VOTRE_USERNAME.github.io/maternelle-rene-cassin-2025/
```

### Accès pour l'équipe :
- Partager le lien GitHub Pages avec l'équipe pédagogique
- Le site est **statique** : aucune installation requise, fonctionne sur n'importe quel navigateur
- Les fiches sont **imprimables** depuis le navigateur (Ctrl+P ou Cmd+P)

---

## 🖨️ Impression des fiches

Les fiches d'activités sont optimisées pour l'impression A4. Pour imprimer :
1. Ouvrir la page souhaitée dans le navigateur
2. Faire Ctrl+P (Windows) ou Cmd+P (Mac)
3. Sélectionner Format A4, marges normales
4. Les styles `print.css` s'appliquent automatiquement
